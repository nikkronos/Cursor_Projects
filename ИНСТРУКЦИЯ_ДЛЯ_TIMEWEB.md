# Инструкция для Timeweb Cloud - как скопировать бота на сервер

## Ваш сервер:
- **Название:** Polite Horologium
- **IP адрес:** 81.200.146.32

## Способ 1: Через веб-консоль Timeweb (САМЫЙ ПРОСТОЙ)

### Шаг 1: Откройте консоль сервера

1. В панели Timeweb нажмите на ваш сервер "Polite Horologium"
2. Перейдите на вкладку **"Консоль"** (вверху есть вкладки: Дашборд, Доступ, Сеть, **Консоль** и т.д.)
3. Откроется веб-консоль в браузере

### Шаг 2: Создайте папку для бота

В консоли выполните команды:

```bash
# Создать папку
mkdir -p /opt/tradetherapybot
cd /opt/tradetherapybot
```

### Шаг 3: Создайте файлы бота прямо в консоли

**Вариант А: Используйте встроенный редактор nano**

1. Создайте файл `main.py`:
```bash
nano main.py
```

2. Скопируйте содержимое файла `main.py` с вашего компьютера и вставьте в nano
3. Сохраните: `Ctrl+O`, `Enter`, `Ctrl+X`

**Повторите для всех файлов:**
- `handlers.py`
- `services.py`
- `database.py`
- `loader.py`
- `config.py`
- `requirements.txt`
- `env_vars.txt`
- `tradetherapybot.service`

**Вариант Б: Используйте wget или curl (если файлы в интернете)**

Если вы загрузите файлы на какой-то хостинг или в git, можно скачать:
```bash
wget https://ваш_адрес/main.py
```

## Способ 2: Через SFTP (FileZilla или WinSCP) - РЕКОМЕНДУЕТСЯ

### Шаг 1: Узнайте данные для подключения

1. В панели Timeweb перейдите на вкладку **"Доступ"** вашего сервера
2. Там будут данные для SSH подключения:
   - **Логин** (обычно `root` или другой пользователь)
   - **Пароль** или **SSH ключ**

### Шаг 2: Установите WinSCP (для Windows)

1. Скачайте WinSCP: https://winscp.net/eng/download.php
2. Установите программу

### Шаг 3: Подключитесь к серверу

1. Откройте WinSCP
2. Заполните данные:
   - **Протокол:** SFTP
   - **Имя хоста:** `81.200.146.32`
   - **Порт:** `22`
   - **Имя пользователя:** (из вкладки "Доступ")
   - **Пароль:** (из вкладки "Доступ")
3. Нажмите "Войти"

### Шаг 4: Скопируйте файлы

1. **Слева** - ваш компьютер, найдите папку:
   ```
   C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\TradeTherapyBot
   ```

2. **Справа** - сервер, перейдите в `/opt/tradetherapybot` (или создайте папку)

3. Выделите все файлы слева (Ctrl+A) и перетащите их вправо

4. Дождитесь завершения копирования

## Способ 3: Через SSH из командной строки Windows

### Шаг 1: Установите OpenSSH (если еще нет)

В Windows 10/11 обычно уже установлен. Проверьте:
```powershell
ssh -V
```

Если не установлен, установите через "Параметры" → "Приложения" → "Дополнительные компоненты"

### Шаг 2: Подключитесь к серверу

Откройте PowerShell или командную строку на вашем компьютере:

```powershell
ssh root@81.200.146.32
```

Или если другой пользователь:
```powershell
ssh ваш_пользователь@81.200.146.32
```

Введите пароль (из вкладки "Доступ" в Timeweb)

### Шаг 3: Создайте папку на сервере

```bash
mkdir -p /opt/tradetherapybot
```

### Шаг 4: Скопируйте файлы через SCP

Откройте **НОВОЕ** окно PowerShell (не закрывайте SSH подключение) и выполните:

```powershell
cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\TradeTherapyBot"

scp -r * root@81.200.146.32:/opt/tradetherapybot/
```

Замените `root` на ваше имя пользователя, если оно другое.

Введите пароль, когда попросит.

## Способ 4: Через веб-файловый менеджер (если есть)

Некоторые панели управления имеют встроенный файловый менеджер. Проверьте в панели Timeweb, есть ли такая функция.

## После копирования файлов:

### 1. Подключитесь к серверу через SSH или консоль

```bash
ssh root@81.200.146.32
```

### 2. Перейдите в папку с ботом

```bash
cd /opt/tradetherapybot
```

### 3. Проверьте, что файлы скопировались

```bash
ls -la
```

Должны быть видны: `main.py`, `handlers.py`, `tradetherapybot.service` и другие.

### 4. Установите зависимости

```bash
python3 --version  # Проверка Python
pip3 install -r requirements.txt
```

### 5. Настройте env_vars.txt

```bash
nano env_vars.txt
```

Убедитесь, что там указаны:
```
BOT_TOKEN=ваш_токен
ADMIN_ID=ваш_id
GROUP_CHAT_ID=id_группы
```

### 6. Настройте systemd (следуйте инструкции из ИНСТРУКЦИЯ_ПО_ЗАПУСКУ.md)

## Какой способ выбрать?

- **Способ 1 (Веб-консоль)** - если файлов немного и вы не хотите устанавливать программы
- **Способ 2 (WinSCP)** - самый удобный для Windows, визуальный интерфейс
- **Способ 3 (SCP)** - если вы знакомы с командной строкой

## Рекомендация:

Используйте **WinSCP (Способ 2)** - это самый простой и наглядный способ для Windows.

## Если не знаете пароль или логин:

1. В панели Timeweb откройте ваш сервер
2. Перейдите на вкладку **"Доступ"**
3. Там будут все данные для подключения
4. Если пароля нет, можно создать нового пользователя или сбросить пароль

## Проблемы?

- **"Connection refused"** - проверьте, что SSH включен в настройках сервера
- **"Permission denied"** - проверьте логин и пароль
- **"Host key verification failed"** - выполните: `ssh-keygen -R 81.200.146.32`
