# 🔄 Как работает автоматический деплой

**Дата:** 02.12.2025

---

## ✅ Ответ на ваш вопрос

**ДА, автоматический деплой работает!** Неважно, откуда вы делаете push:
- ✅ Из командной строки (как вы только что сделали)
- ✅ Через GitHub веб-интерфейс
- ✅ Из любого другого места

**Автоматический деплой сработает в любом случае!**

---

## 🔄 Как это работает (пошагово)

### 1. Вы делаете push в ветку `main`

```
git push origin main
```

**Неважно, откуда:**
- ✅ Из командной строки (PowerShell, Git Bash)
- ✅ Через GitHub веб-интерфейс (редактирование файлов на сайте)
- ✅ Из VS Code, Cursor или другой IDE

### 2. Автоматически запускаются тесты

**GitHub Actions автоматически:**
- Запускает workflow "Run Tests"
- Выполняет все тесты из папки `tests/`
- Проверяет, что всё работает

### 3. Если тесты прошли → автоматически запускается деплой

**GitHub Actions автоматически:**
- Запускает workflow "Deploy to Server"
- Подключается к серверу по SSH
- Создает бэкап базы данных
- Обновляет код на сервере
- Обновляет зависимости
- Перезапускает бота

### 4. Если тесты НЕ прошли → деплой НЕ запускается

**Это защита!** Деплой запускается только после успешных тестов.

---

## 📊 Как проверить, что деплой запустился

### Способ 1: Через GitHub веб-интерфейс

1. Откройте ваш репозиторий на GitHub:
   ```
   https://github.com/nikkronos/TradeTherapyBot
   ```

2. Нажмите на вкладку **"Actions"** (вверху страницы)

3. Вы увидите:
   - ✅ **"Run Tests"** - должен быть зелёный (успешно)
   - ✅ **"Deploy to Server"** - должен быть зелёный (успешно)

### Способ 2: Посмотреть логи

1. В разделе **Actions** нажмите на последний workflow
2. Вы увидите все шаги:
   - Checkout code
   - Setup SSH
   - Backup database
   - Install dependencies
   - Pull latest code
   - Restart service
   - И т.д.

## 📝 Примечание

Если вы хотите запустить деплой БЕЗ тестов (вручную):

1. Откройте GitHub → Actions
2. Выберите workflow "Deploy to Server"
3. Нажмите "Run workflow" (справа вверху)
4. Выберите ветку `main`
5. Нажмите "Run workflow"

Но обычно это не нужно - всё работает автоматически!
