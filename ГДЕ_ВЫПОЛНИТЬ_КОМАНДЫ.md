# 📍 Где выполнить команды для коммита

**Команды нужно выполнить на ВАШЕМ КОМПЬЮТЕРЕ (локально), НЕ на сервере!**

---

## 🖥️ Где выполнять команды

### ✅ На вашем компьютере (Windows)
- Откройте **PowerShell** (или **Git Bash**)
- Выполните команды в папке проекта

### ❌ НЕ на сервере
- Команды `git add`, `git commit`, `git push` выполняются локально

---

## 📝 Пошаговая инструкция

### Шаг 1: Откройте PowerShell

На вашем компьютере:
- Нажмите `Win + R`
- Введите `powershell`
- Нажмите `Enter`

Или:
- Откройте папку проекта в проводнике
- Зажмите `Shift` и нажмите правой кнопкой мыши
- Выберите "Открыть окно PowerShell здесь"

---

### Шаг 2: Перейдите в папку проекта

```powershell
cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\TradeTherapyBot"
```

Или если PowerShell уже открыт в нужной папке - ничего делать не нужно.

---

### Шаг 3: Выполните команды Git

```powershell
# Добавить все файлы
git add .

# Сделать коммит
git commit -m "feat: автоматический CI/CD деплой - завершено, бот работает

- Добавлен GitHub Actions workflow для автоматического деплоя
- Настроен автоматический бэкап БД перед деплоем
- Настроен автоматический перезапуск бота после деплоя
- Бот настроен на сервере (env_vars.txt создан)
- Бот работает и обрабатывает пользователей (03.12.2025)
- Добавлена документация по настройке CI/CD
- Обновлена база знаний
- Созданы файлы для следующего агента

Протестировано: Deploy #4 успешно, бот работает!"

# Отправить в GitHub
git push origin main
```
