# Быстрая настройка боевого бота

## 🚀 Быстрый старт

### 1. Получите токен бота
- Откройте @BotFather в Telegram
- `/mybots` → выберите @tradetherapybot
- Скопируйте **API Token**

### 2. Получите ваш ID
- Отправьте `/me` боту (или используйте @userinfobot)
- Скопируйте ваш ID

### 3. Настройте на сервере

```bash
# Подключитесь к серверу
ssh root@81.200.146.32

# Отредактируйте конфигурацию
cd /opt/tradetherapybot
nano env_vars.txt
```

**Вставьте:**
```
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_ID=ваш_telegram_id
GROUP_CHAT_ID=id_группы_когда_будет_готова
```

**Сохраните:** `Ctrl+O`, `Enter`, `Ctrl+X`

**Перезапустите бота:**
```bash
systemctl restart tradetherapybot.service
systemctl status tradetherapybot.service
```

### 4. Проверьте работу
- Найдите @tradetherapybot в Telegram
- Отправьте `/start`
- Проверьте `/admin` (должно открыться меню)

## ⚠️ Важно: Права администратора

**Без прав администратора в группе НЕ работают:**
- ❌ Автоматическое удаление пользователей
- ❌ Автоматическое одобрение заявок
- ❌ Защита от входа по ссылке

**Работает всё остальное:**
- ✅ Все пользовательские функции
- ✅ Админ-панель
- ✅ Уведомления
- ✅ Проверка подписок

**Рекомендация:** Сделайте бота администратором группы для полной функциональности.
