# Trade Therapy Bot

Бот для управления платной подпиской в закрытое Telegram-сообщество.

## Установка и Запуск

1. **Клонировать репозиторий**
2. **Создать виртуальное окружение:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```
3. **Установить зависимости:**
   ```bash
   pip install -r requirements.txt
   ```
4. **Настроить `env_vars.txt`:**
   Скопируйте файл `env_vars.example.txt` в `env_vars.txt` и заполните переменные:
   ```bash
   cp env_vars.example.txt env_vars.txt
   ```
   Затем отредактируйте `env_vars.txt`:
   ```
   BOT_TOKEN=ваш_токен_бота
   ADMIN_ID=ваш_id_телеграм
   GROUP_CHAT_ID=id_закрытой_группы
   ```
5. **Запуск:**
   
   **Локально (для тестирования):**
   ```bash
   python main.py
   ```
   
   **На сервере (автоматический запуск):**
   См. инструкцию в `README_DEPLOYMENT.md` или `QUICK_START.md`

## Структура проекта

*   `main.py` - Точка входа. Запускает бота и планировщик.
*   `handlers.py` - Обработка всех команд и сообщений от пользователей.
*   `services.py` - Фоновые задачи (проверка подписок, удаление пользователей).
*   `database.py` - Работа с базой данных SQLite (`bot.db`).
*   `loader.py` - Инициализация бота и логгера.
*   `config.py` - Загрузка настроек.

## Команды Админа

*   `/admin` - Главное меню администратора (управление днями, просмотр пользователей, чеков).
*   `/backup` - Получить файл базы данных (`bot.db`) в чат. Рекомендуется делать регулярно.
*   `/set_subscription ...` - Ручная установка подписки (формат см. в коде или справке).
*   `/add_subscription_days ...` - Добавление дней.
*   `/remove_subscription_days ...` - Удаление дней.
*   `/migrate_user <user_id>` - Выдать подписку пользователю до конца месяца (для миграции существующих участников).

## База Данных

Используется **SQLite** (`bot.db`). Файл создается автоматически при первом запуске.
**Важно:** При переустановке сервера не забудьте сохранить этот файл!
