## 🚀 Как это работает теперь

### При каждом изменении кода:

1. **Вы редактируете код** локально
2. **Вы делаете:**
   ```bash
   git add .
   git commit -m "описание"
   git push origin main
   ```
3. **Автоматически:**
   - ✅ Запускаются тесты
   - ✅ Если тесты прошли → код обновляется на сервере
   - ✅ Создается бэкап БД
   - ✅ Обновляются зависимости
   - ✅ Бот перезапускается
