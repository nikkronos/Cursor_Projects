# Проверка работы ноды Code в N8N

## Как проверить, что нода Code работает

### 1. Проверить в редакторе

1. Перейдите на вкладку **"Editor"** (не "Executions")
2. Должны быть видны **три ноды**:
   - Telegram Webhook
   - **Code** (новая нода)
   - Respond to Telegram
3. Ноды должны быть **соединены** стрелками

### 2. Проверить в Executions

1. Отправьте команду `/test` боту
2. Перейдите на вкладку **"Executions"**
3. Кликните на последнее выполнение (самое свежее)
4. Должны быть видны **три ноды** с зелёными галочками:
   - Telegram Webhook ✅
   - **Code** ✅ (должна быть видна!)
   - Respond to Telegram ✅

### 3. Проверить данные в ноде Code

1. Кликните на ноду **Code** в выполнении
2. Должны быть видны:
   - **INPUT** - входящие данные (из webhook)
   - **OUTPUT** - обработанные данные (с полем `message`)

### 4. Если ноды Code нет

**Возможные причины:**
- Нода не была добавлена
- Workflow не был сохранён
- Workflow не был активирован

**Решение:**
1. Перейдите на вкладку **"Editor"**
2. Проверьте, есть ли нода Code между webhook и response
3. Если нет - добавьте её (см. `docs/n8n-add-data-processing.md`)
4. Сохраните workflow (Ctrl+S)
5. Убедитесь, что workflow активирован
