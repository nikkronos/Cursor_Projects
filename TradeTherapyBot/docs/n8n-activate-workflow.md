# Как активировать workflow в N8N

## Где найти кнопку активации

### Способ 1: Тумблер "Active" (основной способ)

1. **Посмотрите в правый верхний угол** экрана
2. Рядом с кнопками "Publish" и "Saved" должен быть **тумблер "Active"**
3. **Переключите тумблер в положение ON** (включено)
   - Когда включен: подсвечен (зелёный/синий)
   - Когда выключен: серый

### Способ 2: Через кнопку "Publish"

1. Нажмите кнопку **"Publish"** (в правом верхнем углу)
2. Это также активирует workflow

### Способ 3: Через меню (три точки)

1. Нажмите на **"..."** (три точки) в правом верхнем углу
2. Найдите опцию **"Activate"** или **"Active"**
3. Включите workflow

## Как проверить, что workflow активирован

### Признаки активированного workflow:

1. ✅ **Кнопка "Deactivate" видна** - если видите эту кнопку, workflow уже активен!
2. ✅ **Webhook URL работает** - тестовый запрос возвращает ответ (например, "OK")
3. ✅ В списке workflows (Overview) workflow отмечен как активный

### Тестирование активации:

1. Скопируйте Production URL из webhook ноды
2. Отправьте тестовый запрос:

```powershell
# В PowerShell
$url = "https://nikkronos.app.n8n.cloud/webhook/telegram-events"
$body = @{event_type="test"; message="Hello N8N!"} | ConvertTo-Json
Invoke-RestMethod -Uri $url -Method Post -Body $body -ContentType "application/json"
```

3. **Если получили ответ "OK"** - workflow активирован и работает! ✅

**Важно:** Если видите кнопку "Deactivate" вместо "Activate" - workflow уже активен, ничего менять не нужно!

## Если не можете найти кнопку активации

### Возможные причины:

1. **Workflow уже активирован** - проверьте статус в списке workflows
2. **Нет прав** - на триале все функции должны быть доступны
3. **Интерфейс изменился** - попробуйте обновить страницу (F5)

### Альтернативный способ:

1. Перейдите в **Overview** (главная страница workflows)
2. Найдите ваш workflow "Telegram Event Webhook Receiver"
3. Рядом с названием должен быть тумблер для активации

## Важно!

⚠️ **Production URL работает только если:**
- Workflow **сохранён** (уже есть - видно "Saved")
- Workflow **активирован** (тумблер "Active" = ON)

Без активации webhook не будет принимать данные!
