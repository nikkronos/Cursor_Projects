## 🔍 Проверка ноды Code

**Важно:** Убедитесь, что нода Code видна в workflow:

1. Перейдите на вкладку **"Editor"** (не "Executions")
2. Должны быть видны **три ноды**:
   - Telegram Webhook
   - **Code** ← должна быть видна!
   - Respond to Telegram

3. Если ноды Code нет:
   - Добавьте её между webhook и response
   - Вставьте код из `docs/n8n-code-fix.md`
   - Сохраните workflow

4. Проверьте в Executions:
   - Отправьте `/test` боту
   - Откройте последнее выполнение
   - Должны быть видны **три ноды** с зелёными галочками
