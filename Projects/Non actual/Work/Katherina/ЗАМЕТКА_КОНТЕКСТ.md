# Катерина — учёт активов и AI-агент (заметка)

**Статус:** пауза до ответа клиента (март 2026).

## Запрос в целом

Телеграм-бот / агент для учёта разных активов и пассивов, обновление минимум раз в неделю и по факту изменений; клиент сообщает агенту — тот заносит и выдаёт анализ.

## Ответы клиента (зафиксировано)

1. **Крипта:** только USDT (Bybit). Не через брокеров (брокерский API не в скоупе).
2. **Анализ:** сводка, рекомендации, отчётность.
3. **Доступ:** 2–3 человека (нужны роли / whitelist Telegram ID).
4. **Долги:** да — кредиты, ипотека, займы и т.п. (чистый капитал = активы − пассивы).
5. **Валюта отчёта:** доллары, рубли, крипта (хранить в нативной валюте, пересчёт по курсам).

## Схема, которую предложил исполнитель

- ТГ-бот: клиент присылает любую релевантную инфу → запись в **БД**, при желании дубль в **таблицу**.
- Выдача: баланс, структура портфеля, простой/расширенный анализ.
- **Автоматически** = подтягивание данных по API (ключи, только чтение где возможно), например Bybit для USDT.
- **Вручную** = клиент пишет «купила / пополнила» и т.д. — бот записывает.
- Опционально: уведомления при событиях, если подключены источники по API.
- Инфра: VPS ~1000 ₽/мес (бот, БД, ключи на сервере) + токены AI (переменная стоимость).
