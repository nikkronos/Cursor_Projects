# Лимиты — виджет пачки лимитных заявок (Т-Инвестиции)

Виджет для массового выставления лимитных заявок через API Т-Инвестиции: выбор инструмента (акции, фьючерсы), ввод суммы в рублях, количества заявок и шага в % — одна кнопка выставляет заявки на покупку ниже текущей цены, вторая — на продажу выше.

## Требования

- Python 3.8+
- Токен Т-Инвестиции (в приложении или [developer.tbank.ru](https://developer.tbank.ru/invest/))

## Установка и запуск локально

```bash
cd Projects/Limits
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
# Создайте env_vars.txt или .env с TINKOFF_INVEST_TOKEN=... и SANDBOX=1
python server.py
```

Откройте в браузере: http://localhost:5000

## Переменные окружения

| Переменная | Описание |
|------------|----------|
| `TINKOFF_INVEST_TOKEN` | Токен API Т-Инвестиции (обязательно) |
| `SANDBOX` | `1` — песочница, иначе боевой контур |
| `PORT` | Порт сервера (по умолчанию 5000) |

Секреты можно положить в `env_vars.txt` или `.env` в корне проекта (не коммитить).

## Деплой на Timeweb 24/7

1. Загрузить проект на сервер (git clone из `nikkronos/Limits` или scp).
2. Настроить виртуальное окружение и зависимости, задать `TINKOFF_INVEST_TOKEN` и при необходимости `SANDBOX=0` в `.env` или в настройках сервиса.
3. Запускать через systemd/gunicorn по аналогии с Damir (см. документацию по Timeweb в репозитории).
