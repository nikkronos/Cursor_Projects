# Исправление ошибки в generate_charts.py

## Проблема
Ошибка `KeyError` на строке 55 при вызове `df.nlargest(10, df['spearman_correlation'].abs())`

## Причина
Метод `nlargest()` ожидает имя колонки (строку), а не Series с вычисленными значениями.

## Решение
Исправлено: сначала создается временная колонка с абсолютными значениями, затем она используется в `nlargest()`:

```python
# Было (неправильно):
top10 = df.nlargest(10, df['spearman_correlation'].abs())

# Стало (правильно):
df['abs_correlation'] = df['spearman_correlation'].abs()
top10 = df.nlargest(10, 'abs_correlation')
```

После использования временная колонка удаляется.

## Статус
✅ Исправлено. Скрипт должен работать корректно.
