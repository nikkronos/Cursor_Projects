**Проблема:** PowerShell имеет проблемы с кириллицей в путях.

**Решение:** Запустите скрипт одним из способов:
- Через командную строку (cmd): `cd путь\к\Version2` → `python generate_charts.py`
- Через двойной клик на `run_generate_charts.bat`
- Через Python IDE (PyCharm, VS Code и т.д.)

**Результат:** Графики будут сохранены в папку `charts/`
