# ИСПРАВЛЕНИЯ В ВЕРСИИ v3

## Исправленные проблемы

### ✅ 1. Верстка титульного листа и содержания
- **Проблема:** Скрипт переформатировал титульный лист и содержание
- **Решение:** Теперь скрипт открывает исходный документ как базу и добавляет к нему только новые разделы (Главу II, выводы, заключение, список литературы, приложения)
- **Результат:** Верстка титульного листа и содержания полностью сохраняется

### ✅ 2. Нумерация в списке литературы
- **Проблема:** Нумерация не отображалась в Word документе
- **Решение:** Убрана строка, которая удаляла нумерацию из markdown файла
- **Результат:** Нумерация (1., 2., 3... 65) теперь сохраняется в Word документе

## Как использовать

1. Запустите `run_merge_v3.bat` (или `python merge_chapters.py` из папки Version2)
2. Откройте созданный файл `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v3.docx`
3. Проверьте:
   - ✅ Титульный лист и содержание не изменены
   - ✅ Нумерация в списке литературы видна (1., 2., 3...)

## Технические детали

- Скрипт теперь открывает исходный Word документ как базу: `doc = Document(chapter1_file)`
- К исходному документу добавляются только новые разделы
- Нумерация в списке литературы сохраняется из markdown файла (не удаляется)
