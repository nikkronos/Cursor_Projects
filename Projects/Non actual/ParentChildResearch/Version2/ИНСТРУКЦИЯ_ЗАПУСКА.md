# Инструкция по запуску скрипта генерации графиков

## Способ 1: Через командную строку (рекомендуется)

1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку Version2:
   ```
   cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\ParentChildResearch\Version2"
   ```
3. Запустите скрипт:
   ```
   python generate_charts.py
   ```

## Способ 2: Через двойной клик на bat-файл

Дважды кликните на файл `run_generate_charts.bat`

## Способ 3: Через Python IDLE или другую IDE

1. Откройте файл `generate_charts.py` в Python IDLE, PyCharm, VS Code или другой IDE
2. Запустите скрипт (F5 или кнопка Run)

## Требования

Убедитесь, что установлены необходимые библиотеки:
```bash
pip install pandas matplotlib numpy seaborn
```

## Результат

После успешного запуска в папке `charts/` будут созданы следующие графики:

1. `1_histogram_correlations.png` - Гистограмма распределения корреляций
2. `2_top10_correlations.png` - Топ-10 самых сильных корреляций
3. `3_significant_correlations.png` - Статистически значимые корреляции
4. `4_comparison_prof_questions.png` - Сравнение по двум вопросам о профессии
5. `5_heatmap_significant.png` - Тепловая карта значимых корреляций (если установлен seaborn)
6. `6_significance_levels.png` - Распределение по уровням значимости

## Если возникают ошибки

1. **Ошибка импорта библиотек**: Установите недостающие библиотеки через `pip install`
2. **Ошибка пути к файлу**: Убедитесь, что файл `correlations_analysis.csv` находится в папке `../newtest/`
3. **Ошибка кодировки**: Убедитесь, что файл CSV сохранен в кодировке UTF-8
