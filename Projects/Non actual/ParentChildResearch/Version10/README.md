# Version10 - Сортировка списка литературы v20 → v30

## Задача
Отсортировать список литературы в документе `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v20.docx` по алфавиту (русские → английские) и обновить все номера источников в тексте.

## Что делает скрипт
1. **Извлекает список литературы** из конца документа v20
2. **Сортирует по алфавиту**: сначала русские источники (А-Я), затем английские
3. **Создаёт маппинг** старых номеров на новые
4. **Обновляет все ссылки** в тексте документа (в параграфах и таблицах)
5. **Обновляет список литературы** в документе с новой нумерацией
6. **Сохраняет результат** как `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v30.docx`

## Запуск

### Вариант 1: Через .bat файл (рекомендуется)
Дважды кликните на файл **`run_fix_v30.bat`**

### Вариант 2: Через командную строку
```bash
cd "ParentChildResearch\Version10"
python fix_v20_to_v30.py
```

## Файлы
- `fix_v20_to_v30.py` - основной скрипт
- `run_fix_v30.bat` - файл для запуска
- `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v20.docx` - исходный документ (автоматически копируется из Version6, если отсутствует)
- `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v30.docx` - результат работы скрипта (создаётся автоматически)
- `СПИСОК_ЛИТЕРАТУРЫ_v30_sorted.md` - отсортированный список литературы в текстовом формате (создаётся для проверки)

## Важно
⚠ **После запуска скрипта обязательно проверьте результат вручную!**

Особенно проверьте:
1. **Правильность сортировки** списка литературы (по алфавиту: русские → английские)
2. **Корректность всех ссылок** в тексте (номера должны соответствовать новой нумерации)
3. **Форматирование списка литературы** в документе (возможно потребуется ручная доработка)

## Особенности реализации
- Скрипт автоматически обрабатывает многострочные записи в библиографии
- Поддерживает различные форматы ссылок: `[N]`, `[N, M]`, `[N; M]`, `[N-M]`
- Сохраняет оригинальные разделители в ссылках (запятые, точки с запятой, дефисы)
- Автоматически копирует документ v20 из Version6, если его нет в Version10

## Если что-то не работает
1. Убедитесь, что установлена библиотека `python-docx`: `pip install python-docx`
2. Убедитесь, что документ `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v20.docx` существует в Version6
3. Проверьте, что в документе есть раздел "СПИСОК ЛИТЕРАТУРЫ"
4. Если есть ошибки - сообщите, какие именно
