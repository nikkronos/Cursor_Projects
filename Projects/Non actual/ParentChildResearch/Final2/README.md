# Инструкция по использованию скриптов анализа корреляций

## Структура файлов

### Основные скрипты:

1. **`comprehensive_analysis.py`** - Перепроверка всех корреляций
   - Загружает данные родителей и подростков
   - Перепроверяет все корреляции между вопросами родителей и вопросами 12 и 19 Овчаровой
   - Сохраняет результаты в `verified_correlations.csv` и `detailed_correlation_data.pkl`

2. **`generate_detailed_calculations.py`** - Генерация полных расчётов
   - Создаёт документ Word с детальными пошаговыми расчётами всех корреляций
   - Формат аналогичен примерам из страниц 7-12 существующего отчёта
   - Результат: `Полные_расчёты_корреляций.docx`

3. **`check_total_scores.py`** - Проверка итоговых результатов
   - Проверяет, использовались ли итоговые баллы по Овчаровой в корреляциях
   - Рассчитывает итоговые баллы для каждого подростка
   - Результат: `итоговые_баллы_овчарова.csv`

4. **`explain_question_selection.py`** - Объяснение выбора вопросов
   - Анализирует статистику по всем 20 вопросам Овчаровой
   - Объясняет выбор вопросов 12 и 19
   - Сравнивает с вопросом 2
   - Результат: `анализ_выбора_вопросов.csv`

5. **`create_final_report.py`** - Создание итогового отчёта
   - Объединяет все результаты в один документ
   - Включает перепроверенные расчёты, полные расчёты, объяснения
   - Оформлен по примеру главы 2.2 из PDF
   - Результат: `Итоговый_отчёт_корреляции.docx`

## Порядок выполнения

### Вариант 1: Автоматический запуск всех скриптов (РЕКОМЕНДУЕТСЯ)

**Просто запустите `run_all.bat`** - он выполнит все скрипты последовательно:
1. Перепроверку корреляций
2. Проверку итоговых результатов
3. Анализ выбора вопросов
4. Создание документа с объяснениями
5. Создание полного итогового отчёта

### Вариант 2: Полный анализ вручную

1. Запустите `run_comprehensive.bat` (или `python comprehensive_analysis.py`)
   - Это создаст файлы с перепроверенными данными

2. Запустите `python check_total_scores.py`
   - Проверит итоговые результаты

3. Запустите `python explain_question_selection.py`
   - Создаст анализ выбора вопросов

4. Запустите `python explanation_document.py`
   - Создаст документ с объяснениями выбора вопросов

5. Запустите `python create_complete_report.py`
   - Создаст полный итоговый отчёт со всеми результатами и полными расчётами

### Вариант 3: Только итоговый отчёт

Запустите `python create_complete_report.py` - он выполнит все необходимые расчёты и создаст отчёт.

## Требования

- Python 3.x
- pandas
- numpy
- scipy
- python-docx

Установка зависимостей:
```bash
pip install pandas numpy scipy python-docx
```

## Результаты

После выполнения всех скриптов будут созданы:

1. `verified_correlations.csv` - перепроверенные корреляции
2. `detailed_correlation_data.pkl` - детальные данные для расчётов
3. `итоговые_баллы_овчарова.csv` - итоговые баллы по Овчаровой
4. `анализ_выбора_вопросов.csv` - статистика по всем вопросам
5. `Обоснование_выбора_вопросов.docx` - документ с объяснениями выбора вопросов 12 и 19
6. **`Полный_итоговый_отчёт_корреляции.docx`** - **ГЛАВНЫЙ ОТЧЁТ** со всеми результатами:
   - Перепроверенными расчётами корреляций
   - Полными пошаговыми расчётами всех корреляций (в формате примеров)
   - Объяснением выбора вопросов 12 и 19
   - Проверкой итоговых результатов
   - Оформлением по примеру главы 2.2 из PDF

## Примечания

- Все скрипты работают с данными из папки `ParentChildResearch/newtest`
- Результаты сохраняются в папке `ParentChildResearch/Final2`
- При возникновении ошибок проверьте, что все необходимые файлы находятся в правильных папках
