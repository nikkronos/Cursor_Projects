# Инструкция по исправлению документа v10 -> v11

## Что исправляет скрипт

### 1. Убирает фамилии из квадратных скобок
**Проблема:** В документе есть ссылки вида `[Ананьев, 3; Кон, 31]` или `[Дружинин, 22; Леонтьев А.Н., 35]`

**Исправление:** Оставляет только номера источников: `[3, 31]`, `[22, 35]`

**Примеры:**
- `[Ананьев, 3; Кон, 31; Дружинин, 22]` → `[3, 31, 22]`
- `[Шнейдер, 63; Эйдемиллер, Юстицкис, 64]` → `[53]` (63 удаляется, 64 → 53)
- `[Выготский, 16; Кле, 29]` → `[16, 29]`

### 2. Добавляет выводы о незначимых связях
**Проблема:** В разделе ВЫВОДЫ есть информация о 5 значимых корреляциях из 122, но нет явного вывода о том, что остальные 117 связей незначимы.

**Исправление:** Добавляется новый пункт выводов (обычно пункт 10), который содержит:
- Информацию о том, что 117 из 122 связей не подтвердились на статистическом уровне
- Объяснение возможных причин отсутствия значимых связей
- Обобщающий вывод о сложности и многогранности процесса профессионального самоопределения

## Запуск

**Дважды кликните на файл `run_fix_v11.bat`**

Или через командную строку:
```bash
python fix_v10_to_v11.py
```

## Результат

Создается документ **ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v11.docx** с исправлениями:
- Все фамилии убраны из квадратных скобок
- Добавлены выводы о незначимых связях

## Важно

После запуска скрипта:
1. **Проверьте документ v11** - убедитесь, что все фамилии убраны из скобок
2. **Проверьте раздел ВЫВОДЫ** - должен быть добавлен пункт о незначимых связях
3. **Проверьте все ссылки** - должны быть только номера в квадратных скобках
