# Version6 - Исправление документа v9 -> v10

## Задача
Исправить проблемы с источниками в документе ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v9.docx:
1. Сверить все номера источников в тексте со списком литературы (1-53)
2. Перефразировать проблемные места с простым перечислением источников:
   - Страница 7: `[Ананьев, 3; Кон, 31; Дружинин, 22]`
   - Страница 10: `[30; 18; 48]`
   - Страница 13: `[Ананьев, 3; Божович, 12; Кон, 31]`
   - Страница 16: `[Выготский, 16; Кле, 29; Кривцова, 32; Кулагина, Колюцкий, 33; Мухина, 44; Ремшмидт, 51]`
3. Создать документ v10 с исправлениями

## Файлы
- `fix_v9_to_v10.py` - основной скрипт для исправления документа
- `analyze_v9.py` - скрипт для анализа документа (поиск всех проблем)
- `extract_text_v9.py` - скрипт для извлечения текста из документа
- `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v9.docx` - исходный документ (из Version5)
- `СПИСОК_ЛИТЕРАТУРЫ_ФИНАЛЬНЫЙ.md` - список литературы (53 источника)

## Запуск

### Вариант 1: Через bat файлы (рекомендуется)

**Основное исправление:**
- Дважды кликните на `run_fix.bat` - это создаст документ v10 с исправлениями

**Дополнительные скрипты:**
- `run_check_text.bat` - проверка текста документа v9
- `run_show_found.bat` - поиск проблемных мест в документе
- `run_compare.bat` - сравнение документов v9 и v10

### Вариант 2: Через командную строку

**Шаг 1: Исправление (основное)**
```bash
python fix_v9_to_v10.py
```
Это создаст документ `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v10.docx` с исправлениями.

**Шаг 2: Анализ (опционально)**
```bash
python analyze_v9.py
```
Это создаст файл `АНАЛИЗ_v9_ОТЧЕТ.txt` с результатами анализа.

## Что исправляет скрипт

1. **Проверка номеров источников**: Удаляет или исправляет номера, которые не входят в диапазон 1-53
2. **Перефразирование проблемных мест**: 
   - `[Ананьев, 3; Кон, 31; Дружинин, 22]` → `как отмечают Ананьев [3], Кон [31] и Дружинин [22]`
   - `[30; 18; 48]` → `как подчеркивают исследователи [18, 30, 48]`
   - И другие известные проблемные места
3. **Автоматическое перефразирование**: Находит и перефразирует перечисления из 3+ авторов в скобках

## Важно

После создания документа v10:
1. **Внимательно проверьте исправленные места** - возможно потребуется ручная доработка
2. **Проверьте все ссылки** - убедитесь, что формат правильный
3. **Проверьте, что все номера источников в диапазоне 1-53**
