## Как использовать

### Генерация графиков

**Вариант 1: Через командную строку (если нет проблем с кириллицей)**
```bash
cd ParentChildResearch/Version1
python generate_charts.py
```

**Вариант 2: Через Python напрямую**
```python
import sys
sys.path.append('ParentChildResearch/Version1')
exec(open('ParentChildResearch/Version1/generate_charts.py').read())
```

**Вариант 3: Запустить из IDE (PyCharm, VS Code и т.д.)**

Графики будут сохранены в папку `charts/`:
- `1_histogram_correlations.png` - Гистограмма распределения корреляций
- `2_top10_correlations.png` - Топ-10 самых сильных корреляций
- `3_significant_correlations.png` - Статистически значимые корреляции
- `4_comparison_prof_questions.png` - Сравнение по двум вопросам о профессии
- `5_heatmap_significant.png` - Тепловая карта (если достаточно данных)
- `6_significance_levels.png` - Распределение по уровням значимости

### Проверка списка литературы

```bash
python check_literature.py
```

Скрипт:
1. Найдет все использованные источники в тексте Главы II
2. Создаст очищенный список литературы
3. Сохранит результат в `Список_литературы_очищенный.txt`

### Анализ значимых корреляций

```bash
python analyze_significant_correlations.py
```

Выведет детальную информацию о всех значимых корреляциях.
