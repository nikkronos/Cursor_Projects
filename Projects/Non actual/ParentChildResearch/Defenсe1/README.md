# Анализ данных по 10 шкалам методики ВРР Марковской

## Описание

Скрипт `vrr_scales_analysis.py` обрабатывает данные опросника "Взаимодействие родитель-ребёнок" (ВРР) И.М. Марковской и рассчитывает баллы по 10 шкалам.

## Структура шкал

1. **Нетребовательность – требовательность** (вопросы: 1, 13*, 25, 37, 49)
2. **Мягкость – строгость** (вопросы: 2, 14*, 26, 38*, 50*)
3. **Автономность – контроль** (вопросы: 3*, 4*, 15, 16, 27, 28, 39*, 40, 51, 52*) - делится на 2
4. **Эмоциональная дистанция – близость** (вопросы: 5, 17, 29, 41*, 53)
5. **Отвержение – принятие** (вопросы: 6*, 7*, 18*, 19*, 30, 31*, 42, 43*, 54*, 55) - делится на 2
6. **Отсутствие сотрудничества – сотрудничество** (вопросы: 8, 20, 32, 44, 56)
7. **Несогласие – согласие** (вопросы: 9*, 21, 33, 45, 57*)
8. **Непоследовательность – последовательность** (вопросы: 10*, 22*, 34*, 46*, 58*)
9. **Авторитетность родителя** (вопросы: 11, 23, 35, 47, 59)
10. **Удовлетворенность отношениями с ребенком** (вопросы: 12, 24, 36, 48*, 60)

*Вопросы, помеченные *, являются обратными и инвертируются перед суммированием.

## Запуск

1. Убедитесь, что файл `Опрос_для_родителей.csv` находится в папке `Defence1`
2. Запустите `run_vrr_analysis.bat` или выполните:
   ```
   python vrr_scales_analysis.py
   ```

## Выходные файлы

После выполнения скрипта будут созданы следующие файлы:

1. **Таблица_сырые_данные_ВРР.docx** - подробная таблица с сырыми данными и промежуточными расчетами для каждого респондента
2. **Сводная_таблица_ВРР.docx** - сводная таблица с описательной статистикой (M, SD, Me, Min, Max) по всем шкалам
3. **Описание_и_интерпретация_ВРР.docx** - подробное описание методики обработки данных и интерпретация результатов

## Методика обработки

### Обратные вопросы
Для обратных вопросов применяется инверсия значений:
- 1 → 5
- 2 → 4
- 3 → 3 (остается без изменений)
- 4 → 2
- 5 → 1

### Расчет итоговых баллов
- Для шкал 1, 2, 4, 6, 7, 8, 9, 10: Итоговый балл = Σ (обработанные значения вопросов)
- Для шкал 3 и 5: Итоговый балл = Σ (обработанные значения вопросов) / 2
