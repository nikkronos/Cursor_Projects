# Инструкция по извлечению текста

## Быстрый способ

Запустите скрипт для извлечения всего текста:

```bash
python extract_all_text.py
```

Это создаст два файла:
1. `ВСЕ_ТЕКСТ_ИЗ_ДОКУМЕНТА.txt` - весь текст документа с номерами параграфов
2. `ПРОБЛЕМНЫЕ_МЕСТА.txt` - только места с фамилиями в скобках

## Альтернативный способ

Если хотите показать мне конкретные проблемные места:

1. Откройте документ v7 в Word
2. Найдите проблемные места (например, страница 4)
3. Скопируйте текст и вставьте сюда

Или:

1. Сохраните проблемные страницы как изображения
2. Я смогу прочитать текст с изображений

## Что делать дальше

После извлечения текста:
1. Откройте `ПРОБЛЕМНЫЕ_МЕСТА.txt`
2. Скопируйте несколько примеров проблемных мест
3. Пришлите мне - я точно настрою паттерны

Или просто запустите `extract_all_text.py` и пришлите содержимое `ПРОБЛЕМНЫЕ_МЕСТА.txt` - я сразу исправлю скрипт.
