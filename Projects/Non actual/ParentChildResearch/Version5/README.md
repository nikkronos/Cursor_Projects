# Version5 - Исправление источников в документе v7

## Задача
Исправить проблемы с источниками в документе ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v7.docx:
1. Убрать фамилии из скобок на странице 4
2. Перефразировать текст с правильным цитированием
3. Проверить соответствие всех номеров источников списку литературы
4. Создать документ v8 с исправлениями

## Файлы
- `analyze_v7.py` - скрипт для анализа документа
- `fix_v7_to_v8.py` - скрипт для исправления проблем
- `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v7.docx` - исходный документ
- `СПИСОК_ЛИТЕРАТУРЫ_ФИНАЛЬНЫЙ.md` - список литературы (53 источника)

## Запуск

### Шаг 1: Анализ
Запустите `START_ANALYSIS.bat` или:
```bash
python analyze_v7.py
```

Это создаст файл `ANALYSIS_RESULTS.txt` с результатами анализа.

### Шаг 2: Исправление
После анализа запустите:
```bash
python fix_v7_to_v8.py
```

Это создаст документ `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v8.docx` с исправлениями.
