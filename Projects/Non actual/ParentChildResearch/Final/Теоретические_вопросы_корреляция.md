# Теоретические вопросы по методам статистического анализа

## 1. Почему использовались методы Пирсона и Спирмена?

### Корреляция Пирсона (r)

**Назначение:** Измеряет линейную связь между двумя непрерывными переменными.

**Применение:**
- Когда данные имеют нормальное распределение (или близкое к нему)
- Когда связь между переменными предположительно линейная
- Когда шкала измерения интервальная или рациональная

**Обоснование использования:**
1. **Шкала опроса:** В опросниках родительско-детских отношений используется шкала Лайкерта (обычно 1-5), которая может рассматриваться как квази-интервальная
2. **Линейная связь:** Корреляция Пирсона подходит для обнаружения линейных зависимостей между показателями стилей воспитания и ответами подростков
3. **Нормальность распределения:** При достаточном размере выборки (n>30) распределение выборочных средних стремится к нормальному (ЦПТ), что позволяет использовать параметрические методы

**Могли ли они не подойти?**
Да, если:
- Распределение данных сильно отличается от нормального
- Связь нелинейная (например, U-образная)
- Есть выбросы, сильно искажающие корреляцию
- Размер выборки очень мал (n<30)

---

### Корреляция Спирмена (ρ, rho)

**Назначение:** Измеряет монотонную связь между двумя переменными (не обязательно линейную).

**Применение:**
- Когда данные имеют порядковую шкалу
- Когда распределение не нормальное
- Когда есть выбросы
- Когда связь монотонная, но не обязательно линейная

**Обоснование использования:**
1. **Порядковая шкала:** Метод Спирмена основан на рангах, что делает его устойчивым к отклонениям от нормальности
2. **Робастность:** Менее чувствителен к выбросам по сравнению с корреляцией Пирсона
3. **Универсальность:** Подходит для всех типов монотонных связей (линейных и нелинейных)
4. **Шкала Лайкерта:** Поскольку шкала Лайкерта может рассматриваться как порядковая, метод Спирмена более корректен с теоретической точки зрения

**Почему использовали оба метода?**
- **Пирсон:** Для проверки линейной связи при условии нормальности
- **Спирмен:** Для проверки монотонной связи без предположения о нормальности
- **Сравнение результатов:** Согласованность результатов обоих методов увеличивает надежность выводов

**Могли ли они не подойти?**
Метод Спирмена подходит практически всегда, когда нужна корреляция. Однако:
- Если связь немонотонная (например, U-образная), оба метода могут показать низкую корреляцию
- При очень малой выборке (n<10) результаты могут быть ненадежными

## 2. Почему не использовался тест Манна-Уитни? Можно ли его использовать?

### Тест Манна-Уитни (U-тест)

**Назначение:** Сравнивает две независимые выборки (проверяет гипотезу о равенстве распределений или медиан).

**Когда используется:**
- Сравнение двух групп по одному показателю
- Когда данные не имеют нормального распределения
- Когда шкала измерения порядковая или интервальная
- Непараметрическая альтернатива t-тесту Стьюдента

**Почему НЕ использовался:**
1. **Разные задачи:**
   - **Корреляция (Пирсон/Спирмен):** Ищет связь между двумя переменными в одной группе
   - **Манна-Уитни:** Сравнивает одну переменную между двумя независимыми группами

2. **Цель исследования:**
   - Исследование родительско-детских отношений направлено на поиск **связей** между ответами родителей и детей
   - Манна-Уитни используется для **сравнения групп** (например, мальчики vs девочки, разные возрастные группы)

3. **Структура данных:**
   - Данные представлены в виде пар "родитель-ребенок"
   - Нужна оценка связи между переменными, а не сравнение групп

**Можно ли использовать тест Манна-Уитни?**
Да, но для других целей:

**Примеры применения:**
1. **Сравнение групп родителей:**
   - Группа 1: Родители мальчиков
   - Группа 2: Родители девочек
   - Показатель: Уровень авторитарности (по опроснику)
   - Гипотеза: Различается ли уровень авторитарности между родителями мальчиков и девочек?

2. **Сравнение групп подростков:**
   - Группа 1: Подростки 14-15 лет
   - Группа 2: Подростки 16-17 лет
   - Показатель: Уровень протестного поведения
   - Гипотеза: Различается ли уровень протестного поведения в разных возрастных группах?

3. **Сравнение стилей воспитания:**
   - Группа 1: Родители с высоким уровнем контроля
   - Группа 2: Родители с низким уровнем контроля
   - Показатель: Успеваемость детей
   - Гипотеза: Влияет ли уровень контроля на успеваемость?

**Когда Манна-Уитни был бы уместен вместо корреляции:**
- Если бы сравнивались независимые группы (не пары родитель-ребенок)
- Если бы проверялась гипотеза о различиях, а не о связи
- Если бы данные были независимыми (например, случайная выборка родителей и случайная выборка детей, не связанных между собой)

**Вывод:**
- **Корреляция (Пирсон/Спирмен)** — для оценки связи между переменными в парных данных
- **Манна-Уитни** — для сравнения двух независимых групп по одному показателю

Оба метода могут использоваться в одном исследовании, но для разных целей и разных гипотез.

## Сравнительная таблица методов

| Метод | Назначение | Тип данных | Когда использовать |
|-------|------------|------------|-------------------|
| **Корреляция Пирсона** | Линейная связь | Интервальная/рациональная, нормальное распределение | Оценка линейной связи между переменными |
| **Корреляция Спирмена** | Монотонная связь | Порядковая, любое распределение | Оценка связи при отклонении от нормальности |
| **Манна-Уитни** | Сравнение групп | Порядковая/интервальная, независимые группы | Сравнение двух независимых групп |

---

## Рекомендации для исследования

1. **Для оценки связи между ответами родителей и детей:**
   - ✅ Корреляция Пирсона (если распределение нормальное)
   - ✅ Корреляция Спирмена (как более надежный вариант)
   - ❌ Манна-Уитни (не подходит для парных данных)

2. **Для сравнения групп (если нужно):**
   - ✅ Манна-Уитни (для непараметрического сравнения)
   - ✅ t-тест Стьюдента (если распределение нормальное)

3. **Комплексный анализ:**
   - Корреляция для оценки связей
   - Манна-Уитни для сравнения подгрупп (например, по полу, возрасту)
   - Регрессионный анализ для предсказания (если нужно)
