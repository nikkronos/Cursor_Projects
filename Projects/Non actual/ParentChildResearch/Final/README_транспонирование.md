# Инструкция по транспонированию CSV файлов

## Описание задачи

Требуется транспонировать две таблицы с ответами на опросы:
1. `Опрос для родителей  (Ответы) - Ответы на форму (1).csv`
2. `Опрос ученика (Ответы) - Ответы на форму (1).csv`

**Что делается:**
- Строки становятся столбцами, столбцы - строками
- Удаляется столбец "Отметка времени"
- Сохраняются остальные метаданные (ФИО, возраст, класс и т.д.)
- В заголовках столбцов используются имена респондентов

**Результат:**
- `Родители трансп.csv` - транспонированная таблица родителей
- `Подростки трансп.csv` - транспонированная таблица подростков

## Скрипты

### 1. `transpose_csv.py` (основной скрипт)
Полнофункциональный скрипт с функциями для каждого типа данных.

Упрощённая версия для быстрого выполнения.

### 3. `run_transpose.bat` (bat-файл для запуска)
Файл для запуска из командной строки Windows.

## Как запустить

### Вариант 1: Через bat-файл (рекомендуется)

1. Откройте папку `ParentChildResearch/Final` в проводнике Windows
2. Дважды щёлкните на файл `run_transpose.bat`
3. Дождитесь завершения выполнения

### Вариант 2: Через командную строку

1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку:
   ```cmd
   cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\ParentChildResearch\Final"
   ```
3. Запустите скрипт:
   ```cmd
   python run_transpose_simple.py
   ```
   или
   ```cmd
   python transpose_csv.py
   ```

### Вариант 3: Через Python напрямую

Если есть проблемы с путями:

1. Откройте Python
2. Выполните:
   ```python
   import os
   os.chdir(r'C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\ParentChildResearch\Final')
   exec(open('run_transpose_simple.py', encoding='utf-8').read())
   ```

## Требования

- Python 3.x
- Библиотека pandas (установить через `pip install pandas`)

---

## Что делают скрипты

1. **Читают CSV файлы** с правильной кодировкой (UTF-8)
2. **Удаляют столбец "Отметка времени"**
3. **Транспонируют таблицы** (меняют строки и столбцы местами)
4. **Используют имена респондентов** в качестве заголовков столбцов
5. **Сохраняют результаты** в новые CSV файлы с кодировкой UTF-8-sig (для корректного отображения в Excel)

---

## Структура результата

После транспонирования:

**Первая колонка:** Названия вопросов/метаданных
**Остальные колонки:** Ответы каждого респондента (заголовки - имена респондентов)

Пример:
```
Вопрос/Метаданные | Новикова Элина | Иванова Василиса | ...
Фамилия, имя родителя | Новикова Элина | Иванова Василиса | ...
Фамилия и имя ребенка | Новиков Егор | Иванов Матвей | ...
Возраст ребенка | 15 | 14 | ...
Вопрос 1 | 3 | 4 | ...
Вопрос 2 | 1 | 4 | ...
...
```

---

## Примечания

- Все файлы создаются в той же папке (`Final`)
- Исходные файлы не изменяются
- Если файлы с результатами уже существуют, они будут перезаписаны
- При возникновении ошибок проверьте, что файлы исходных данных находятся в той же папке
