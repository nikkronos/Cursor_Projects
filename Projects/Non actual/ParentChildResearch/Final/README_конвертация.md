# Инструкция по конвертации Markdown в Word

## Описание

Скрипт для конвертации файла `Теоретические_вопросы_корреляция.md` в Word документ (.docx).

## Как запустить

### Вариант 1: Через bat-файл (рекомендуется)

1. Откройте папку `ParentChildResearch/Final` в проводнике Windows
2. Дважды щёлкните на файл `convert_to_word.bat`
3. Дождитесь завершения выполнения

### Вариант 2: Через командную строку

1. Откройте командную строку (cmd)
2. Перейдите в папку:
   ```cmd
   cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\ParentChildResearch\Final"
   ```
3. Запустите скрипт:
   ```cmd
   python convert_md_to_word.py
   ```

## Результат

После выполнения будет создан файл:
- `Теоретические_вопросы_корреляция.docx`

Файл будет содержать:
- Заголовки разных уровней
- Форматированный текст (жирный, моноширинный)
- Таблицы
- Списки (нумерованные и маркированные)
- Разделители

## Требования

- Python 3.x
- Библиотека python-docx (установить через `pip install python-docx`)
