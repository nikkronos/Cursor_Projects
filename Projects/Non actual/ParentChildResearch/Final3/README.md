# Улучшенный анализ корреляций - Final3

## Описание

Этот скрипт создаёт подробный анализ корреляций с:
- Подробными расчётами (M, SD, Me, %)
- Объяснением "Уровень значимости" простым языком
- Таблицами для каждой значимой связи
- Визуализациями:
  - Столбчатые диаграммы по 6 типам Холланда с error bars (SD)
  - Тепловая карта значимых корреляций
  - Столбчатая диаграмма по мотивам
  - Мини-матрица корреляций между Овчаровой и Марковской

## Требования

Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

Просто запустите:
```bash
run_analysis.bat
```

Или напрямую:
```bash
python enhanced_analysis.py
```

## Результаты

После выполнения будут созданы:

1. **Подробный_анализ_корреляций.docx** - документ Word с:
   - Объяснением уровня значимости
   - Объяснением статистических показателей (M, SD, Me, %)
   - Таблицами для каждой значимой связи
   - Визуализациями

2. **Папка graphs/** с графиками:
   - `holland_types_bar_chart.png` - столбчатая диаграмма по типам Холланда
   - `ovcharova_holland_heatmap.png` - тепловая карта Овчарова-Холланд
   - `markovskaya_holland_heatmap.png` - тепловая карта Марковская-Холланд
   - `motives_bar_chart.png` - столбчатая диаграмма по мотивам
   - `ovcharova_markovskaya_mini_matrix.png` - мини-матрица корреляций

## Структура данных

Скрипт работает с данными из папки `../newtest/`:
- `Опрос для родителей  (Ответы).csv` - данные родителей (ВРР Марковской)
- `Опрос ученика (Ответы) Новый.csv` - данные подростков (Овчарова + Холланд)

## Статистические показатели

- **M (mean)** - среднее арифметическое
- **SD (standard deviation)** - стандартное отклонение
- **Me (median)** - медиана
- **% (проценты)** - доля респондентов в процентах

## Уровень значимости

- **p < 0.001 (***)** - очень высокая значимость
- **p < 0.01 (**)** - высокая значимость
- **p < 0.05 (*)** - значимость
- **p ≥ 0.05** - незначимо
