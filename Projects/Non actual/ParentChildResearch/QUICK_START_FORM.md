# Быстрый старт: Заполнение Google Forms

## Важно: Два разных скрипта!

### 1. `analyze_form_structure.py` - ТОЛЬКО АНАЛИЗ
Этот скрипт **НЕ заполняет** форму, он только анализирует её структуру и сохраняет HTML.
- Используется для отладки и понимания структуры формы
- Сохраняет HTML каждой страницы
- Анализирует паттерны организации вопросов

### 2. `submit_to_google_form.py` - ЗАПОЛНЕНИЕ ФОРМЫ
Этот скрипт **заполняет и отправляет** данные из CSV в форму.
- Читает данные из CSV файла
- Заполняет все поля формы
- Отправляет форму

## Как заполнить форму

### Шаг 1: Подготовка CSV файла
Убедитесь, что CSV файл находится по пути:
```
c:\Users\krono\Downloads\new-all_50_pairs.csv
```

Или измените путь в файле `scripts/submit_to_google_form.py`:
```python
csv_file = r'ваш\путь\к\файлу.csv'
```

### Шаг 2: Запуск скрипта заполнения

**Вариант 1: Через bat-файл**
```powershell
.\run_submit_form.bat
```

**Вариант 2: Напрямую через Python**
```powershell
python scripts\submit_to_google_form.py
```

### Шаг 3: Тестовый режим
По умолчанию скрипт работает в **тестовом режиме** (`test_mode=True`), обрабатывая только первую запись.

Для обработки всех записей откройте `scripts/submit_to_google_form.py` и измените:
```python
test_mode = False  # Изменить с True на False
```

## Что происходит при запуске

1. Скрипт открывает Chrome браузер
2. Переходит на форму Google Forms
3. Заполняет страницу 1 (Имя, Возраст, Класс)
4. Заполняет страницу 2 (60 вопросов Q1-Q60)
5. Заполняет страницу 3 (42 вопроса Choice1-Choice42)
6. Заполняет страницу 4 (20 вопросов ProfQ1-ProfQ20)
7. Отправляет форму

## Логи и отладка

### Логи
Все действия логируются в файл:
```
logs/form_submission_YYYYMMDD_HHMMSS.log
```

### Прогресс
Прогресс сохраняется в:
```
data/generated/submission_progress.json
```

### Скриншоты при ошибках
Скриншоты сохраняются в:
```
data/generated/screenshots/
```

## Устранение проблем

### Скрипт не находит радиокнопки
1. Проверьте логи в `logs/` для детальной информации
2. Убедитесь, что форма открывается корректно
3. Попробуйте увеличить задержку (`delay` параметр)

### Радиокнопки не кликаются
1. Скрипт использует 4 стратегии поиска радиокнопок
2. Если одна не работает, автоматически пробуется следующая
3. Проверьте логи, какая стратегия сработала

### Возобновление после ошибки
Откройте `data/generated/submission_progress.json` и найдите `last_processed_index`.
Затем в скрипте установите:
```python
start_index = last_processed_index + 1
```
