# ИНСТРУКЦИЯ ПО ЗАПУСКУ СКРИПТОВ

## Шаг 1: Установка зависимостей
```bash
pip install python-docx
```

## Шаг 2: Запуск основного скрипта исправлений
```bash
python FINAL_FIX_SCRIPT.py
```

Этот скрипт выполнит все автоматические исправления (А, Б, Г, Е, И, Й, К, М) и создаст документ v7.

## Шаг 3: Анализ литературы
```bash
python analyze_literature.py
```

Этот скрипт:
- Проанализирует использование источников в тексте
- Сформирует актуальный список литературы
- Покажет, какие источники используются, а какие нет
- Проверит, укладываемся ли в 50 источников

## Шаг 4: Исправление цитирования (опционально)
```bash
python fix_citations.py
```

Этот скрипт заменит простые перечисления источников на цитирование с указанием авторов.

## Результаты

После выполнения скриптов будут созданы:
- `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v7.docx` - исправленный документ
- `СПИСОК_ЛИТЕРАТУРЫ_АКТУАЛЬНЫЙ.md` - актуальный список литературы
