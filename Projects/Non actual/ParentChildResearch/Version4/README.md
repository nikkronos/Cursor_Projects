# Version4 - Работа с ВКР v6 -> v7

## Запуск скрипта извлечения текста

Запустите в терминале:
```bash
python work_with_docx.py
```

Или через bat-файл:
```bash
run_extract.bat
```

Скрипт:
1. Скопирует необходимые файлы из Version3 и Version2
2. Извлечёт текст из документов
3. Сохранит в v6_text.txt и source_text.txt

## Установка зависимостей

```bash
pip install python-docx
```
