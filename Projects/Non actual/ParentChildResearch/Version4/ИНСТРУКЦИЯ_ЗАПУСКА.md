# ИНСТРУКЦИЯ ПО ЗАПУСКУ

## Выполнение всех задач по порядку

### Вариант 1: Автоматический (рекомендуется)

Запустите файл: **`run_all.bat`**

Этот файл выполнит все задачи последовательно:
1. Задача В - добавление фамилий к подходам
2. Задачи Д, З, Н, О - цитирование
3. Работа со списком литературы

### Вариант 2: Вручную через Python

```bash
cd ParentChildResearch\Version4
python execute_all_tasks.py
```

### Вариант 3: По отдельности

Если нужно выполнить задачи по отдельности:

1. **Задача В:**
   ```bash
   python fix_task_V.py
   ```

2. **Задачи Д, З, Н, О:**
   ```bash
   python fix_citations_D_З_Н_О.py
   ```

3. **Список литературы:**
   ```bash
   python fix_literature_list.py
   ```

## Результаты

После выполнения будут обновлены:
- `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v7.docx` - основной документ
- `СПИСОК_ЛИТЕРАТУРЫ_ФИНАЛЬНЫЙ.md` - финальный список литературы

## Примечания

- Убедитесь, что установлен `python-docx`: `pip install python-docx`
- Все скрипты работают с документом `ВКР_ПОЛНЫЙ_ДОКУМЕНТ_v7.docx`
- Задачи Ж и П отменены и не выполняются
