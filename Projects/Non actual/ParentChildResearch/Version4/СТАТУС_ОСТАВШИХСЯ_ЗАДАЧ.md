# Статус оставшихся задач

## ✅ Готово к выполнению

### Задача В - Добавление фамилий к подходам
**Скрипт:** `fix_task_V.py`

**Что делает:**
- Добавляет фамилии авторов к первым трём подходам на странице 4
- Формат: `[Автор, номер; Автор, номер]` (как в задаче Й)

**Результат:**
- `Системный семейный подход [Андреева, 5; Эйдемиллер, Юстицкис, 64]`
- `Психологические теории развития и сценариев [Адлер; 2]`
- `Теории профессионального самоопределения [Гинзбург, 18; Климов, 30; Прихожан, Толстых, 49]`

### Задачи Д, Ж, З, Н, О, П - Цитирование
**Скрипт:** `fix_citations_manual.py`

**Что делает:**
- Заменяет перечисления источников на цитирование с указанием авторов
- Формат: `[Автор, номер; Автор, номер]`

**Места для исправления:**
- Строка 66: `[3; 31; 22]` → `[Ананьев, 3; Кон, 31; Дружинин, 22]`
- Строка 168: `[22; 35]` → `[Дружинин, 22; Леонтьев А.Н., 35]`
- Строка 174: `[20; 21]` → `[Деркач, Зазыкин, 20; Дружилов, 21]`
- Строка 174: `[26; 42]` → `[Иванников, 26; Мерлин, 42]`
- Строка 214: `[3; 4]` → `[Ананьев, 3; Ананьев, 4]`
- Строка 215: `[20; 21; 26]` → `[Деркач, Зазыкин, 20; Дружилов, 21; Иванников, 26]`
- Строка 218: `[1; 2; 7; 9; 34; 41; 54]` → с фамилиями
- Строка 218: `[18; 27; 30; 48]` → с фамилиями

## 🚀 Запуск

Запустите:
```bash
run_remaining_tasks.bat
```

Или по отдельности:
```bash
python fix_task_V.py
python fix_citations_manual.py
```

## ⚠ Примечания

1. **Задача В:** Фамилии берутся из списка литературы. Если в исходнике указаны другие авторы, нужно будет скорректировать вручную.

2. **Задачи Д-П:** Скрипт исправляет основные места, но могут быть другие места с перечислениями, которые нужно проверить вручную.

3. **Проверка:** После выполнения скриптов рекомендуется проверить документ v7 и убедиться, что все исправления применены корректно.
