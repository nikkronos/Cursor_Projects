# Инструкция по запуску скриптов

## Шаг 1: Установка зависимостей

```bash
pip install python-docx
```

## Шаг 2: Извлечение текста из документов

Запустите:
```bash
python execute_extraction.py
```

Или:
```bash
python quick_extract.py
```

Это скопирует файлы из Version3 и Version2, и извлечёт текст в `v6_text.txt`.

## Шаг 3: Анализ и правки

После извлечения текста будет создан файл `v6_text.txt`, который можно проанализировать для выполнения всех задач А-П.
