# Выполненные задачи — проект «Шахматы»

## Сессия 2026-02-05

**Дата:** 2026-02-05  

**Выполнено:**

- Создана папка `Chess` с отдельным небольшим веб‑проектом.
- Реализован сервер на `aiohttp` с хранением состояния доски и раздачей ролей игрокам.
- Реализовано подключение нескольких клиентов по WebSocket и синхронизация ходов.
- Реализован клиент на HTML/CSS/JavaScript с доской 14×14 и четырьмя цветами фигур.
- Реализована базовая валидация ходов на сервере (фигуры ходят «по‑шахматному», но без шаха/мата).
- Добавлена документация по запуску (`README.md`) и базовый roadmap проекта.

**Результат:**  
Локально можно запустить сервер и сыграть в простую версию шахмат на четверых с друзьями через браузер.

### Доработки (шах/мат, подсветка, русские названия)

**Дата:** 2026-02-05

**Выполнено:**

- Добавлена полная логика шаха и мата:
  - проверка шаха для каждого игрока;
  - запрет ходов, оставляющих своего короля под шахом;
  - автоматическое завершение игры при мате;
  - отслеживание выбывших игроков.
- Добавлена подсветка возможных ходов:
  - при выборе фигуры запрашиваются возможные ходы у сервера;
  - подсветка доступных клеток жёлтым цветом.
- Изменены названия фигур на русские: Король (Ко), Ферзь (Ф), Конь (К), Слон (С), Ладья (Л).
- Улучшен UX: показ статуса "ШАХ!", победителя, автоматический сброс выбора.
- Добавлено автоматическое определение IP адресов (локальный и публичный) при запуске сервера.

**Результат:**  
Полнофункциональная игра в шахматы на четверых с проверкой шаха/мата, подсветкой ходов и русскими названиями фигур. Готова для игры через интернет.
