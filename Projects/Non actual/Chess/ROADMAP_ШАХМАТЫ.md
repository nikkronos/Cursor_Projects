# Roadmap проекта «Шахматы»

## Завершено ✅

- [x] Создан отдельный проект `Chess` с сервером и веб‑клиентом.
- [x] Реализована базовая 4‑игроковая доска 14×14 с четырьмя цветами.
- [x] Реализован обмен ходами по WebSocket (один сервер → несколько клиентов).
- [x] Реализованы базовые правила ходов фигур (без шаха/мата).
