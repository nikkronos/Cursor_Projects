# Шахматы на четверых

Простой веб‑проект «Шахматы» для игры в четыре игрока по сети.

- Сервер на Python (`aiohttp`, WebSocket).
- Клиент — одна HTML‑страница с JavaScript.
- Играть можно локально (несколько вкладок/браузеров) или по сети, если пробросить порт сервера.

## Как запустить

1. Установите зависимости (рекомендуется виртуальное окружение):

```bash
cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\Chess"
pip install -r requirements.txt
```

2. Запустите сервер:

```bash
python server.py
```

3. Откройте в браузере (на этой же машине или по сети):

- `http://localhost:8000/` — первая вкладка/первый игрок
- `http://<IP_СЕРВЕРА>:8000/` — остальные игроки по сети

Сервер автоматически покажет ваш локальный и публичный IP при запуске.

**Для игры через интернет (БЕЗ проброса порта):**

### Вариант 1: Localtunnel (рекомендуется, не требует регистрации)

1. Установите Node.js с https://nodejs.org/ (если ещё не установлен)
2. Установите localtunnel:
   ```bash
   npm install -g localtunnel
   ```
3. В одном окне терминала запустите туннель:
   ```bash
   npx localtunnel --port 8000
   ```
   или если установили глобально:
   ```bash
   lt --port 8000
   ```
   Скопируйте появившийся URL (например, `https://xxxxx.loca.lt`)
4. В другом окне запустите сервер:
   ```bash
   python server.py
   ```
5. Друзья подключаются по URL из шага 3

### Вариант 2: Ngrok (требует бесплатную регистрацию)

1. Скачайте ngrok с https://ngrok.com/download
2. Зарегистрируйтесь на https://dashboard.ngrok.com/ и получите токен
3. Настройте токен: `ngrok config add-authtoken <ВАШ_ТОКЕН>`
4. В одном окне запустите: `ngrok http 8000`
5. В другом окне запустите: `python server.py`
6. Откройте http://127.0.0.1:4040 чтобы увидеть публичный URL
7. Друзья подключаются по этому URL

### Вариант 3: Проброс порта (если туннели не подходят)

1. Настройте проброс порта 8000 на вашем роутере (port forwarding)
2. Разрешите порт 8000 в файрволе Windows
3. Друзья подключаются по адресу: `http://<ВАШ_ПУБЛИЧНЫЙ_IP>:8000/`

Сервер раздаёт цвета в порядке подключений: **красный → синий → жёлтый → зелёный**.
Пятый и последующие клиенты становятся зрителями.

## Возможности

- ✅ **Шах и мат**: полная проверка шаха и мата, игра завершается при мате
- ✅ **Подсветка возможных ходов**: при выборе фигуры подсвечиваются все доступные ходы
- ✅ **Русские названия фигур**: Король (Ко), Ферзь (Ф), Конь (К), Слон (С), Ладья (Л)
- ✅ **Базовые правила**: все фигуры ходят по классическим шахматным правилам
- ✅ **Промоция пешек**: пешки превращаются в ферзя при достижении дальней стороны

## Ограничения

- Нет рокировки, взятия на проходе и прочих продвинутых правил
- Логика очков/команд (FFA/Teams) не реализована — игра заканчивается при мате одного из игроков

## Структура папки `Chess`

- `server.py` — сервер aiohttp (HTTP + WebSocket, хранит состояние доски).
- `requirements.txt` — зависимости только для проекта «Шахматы».
- `index.html` — главная страница.
- `static/style.css` — стили.
- `static/game.js` — клиентская логика (отрисовка доски, взаимодействие по WebSocket).
- `ROADMAP_ШАХМАТЫ.md` — планы по развитию проекта.
- `DONE_LIST_ШАХМАТЫ.md` — выполненные задачи.
- `SESSION_SUMMARY_2026-02-05.md` — текущая сессия.
