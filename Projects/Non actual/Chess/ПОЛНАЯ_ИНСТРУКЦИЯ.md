# Полная инструкция: игра в шахматы на четверых через интернет

## 🎮 Для вас (хост-сервер)

### Шаг 1: Установка зависимостей Python (только один раз!)

**⚠️ ВАЖНО:** Эту команду нужно выполнить только один раз, не каждый раз!

Откройте PowerShell и выполните:

```powershell
cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\Chess"
pip install -r requirements.txt
```

После успешной установки эту команду больше выполнять не нужно!

### Шаг 2: Запуск сервера

В **первом окне PowerShell** выполните:

```powershell
cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\Chess"
python server.py
```

Вы увидите:
```
============================================================
Сервер запущен!
Локальный IP: 192.168.x.x
Публичный IP: x.x.x.x
============================================================
```

**Оставьте это окно открытым!** Сервер должен работать всё время игры.

### Шаг 3: Запуск туннеля (для игры через интернет)

Откройте **второе окно PowerShell** и выполните:

```powershell
npx.cmd localtunnel --port 8000
```

Вы увидите:
```
your url is: https://xxxxx.loca.lt
```

**Скопируйте этот URL!** Это адрес, по которому друзья будут подключаться.

**⚠️ ВАЖНО:** Localtunnel теперь требует пароль! Пароль = ваш публичный IP адрес.

**Как получить пароль:**
- В окне сервера вы увидите "Публичный IP: x.x.x.x" — это и есть пароль
- Или откройте в браузере: `https://loca.lt/mytunnelpassword`

**Отправьте друзьям:**
1. URL туннеля (например, `https://tangy-ducks-leave.loca.lt`)
2. Пароль (ваш публичный IP, например, `94.19.223.132`)

**⚠️ Важно:** Оба окна (сервер и туннель) должны быть открыты всё время игры!

### Шаг 4: Подключение к игре

Откройте в браузере:
- `http://localhost:8000/` — для вас на этой машине
- Или используйте URL из шага 3 (например, `https://xxxxx.loca.lt`)

## 👥 Для ваших друзей

### Шаг 1: Получите от вас URL и пароль

Попросите у вас:
- URL туннеля (например, `https://tangy-ducks-leave.loca.lt`)
- Пароль туннеля (это публичный IP адрес, например, `94.19.223.132`)

### Шаг 2: Откройте в браузере

1. Откройте URL в любом браузере (Chrome, Firefox, Edge, Safari)
2. Вы увидите страницу с запросом пароля
3. Введите пароль (публичный IP адрес, который вам дали)
4. Нажмите "Click to Submit"

### Шаг 3: Играйте!

- Первые 4 подключившихся получат цвета: **красный → синий → жёлтый → зелёный**
- Пятый и дальше станут зрителями
- Нажмите на свою фигуру, затем на клетку, куда хотите походить

## ⚠️ Важные моменты

1. **Оба окна должны быть открыты** — и сервер, и туннель
2. **URL туннеля меняется** при каждом запуске — отправляйте новый URL друзьям каждый раз
3. **Туннель работает только пока запущен** — если закрыть окно с туннелем, друзья потеряют подключение
4. **Первый игрок (красный) ходит первым** — ходы идут по часовой стрелке

## 🆘 Если что-то не работает

### Сервер не запускается:
- Проверьте, что установлены зависимости: `pip install -r requirements.txt`
- Убедитесь, что порт 8000 свободен (закройте другие программы, использующие этот порт)

### Туннель не запускается:
- Проверьте, что Node.js установлен: `node --version`
- Проверьте, что localtunnel установлен: `npm.cmd list -g localtunnel`
- Если не установлен: `npm.cmd install -g localtunnel`

### Друзья не могут подключиться:
- Убедитесь, что оба окна (сервер и туннель) открыты
- Проверьте, что URL правильный (скопируйте полностью)
- Попросите друзей обновить страницу (F5)
