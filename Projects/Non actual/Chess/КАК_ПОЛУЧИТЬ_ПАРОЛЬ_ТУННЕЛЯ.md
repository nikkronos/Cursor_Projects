# Как получить пароль туннеля для localtunnel

## Проблема
Localtunnel теперь требует пароль для доступа. Пароль = ваш публичный IP адрес.

## Решение: получить пароль

### Способ 1: Через браузер (самый простой)

На вашем компьютере (где запущен туннель) откройте в браузере:

```
https://loca.lt/mytunnelpassword
```

Вы увидите ваш публичный IP адрес — это и есть пароль туннеля.

**Пример:** Если вы видите `94.19.223.132`, то это и есть пароль.

### Способ 2: Через командную строку

В PowerShell выполните:

```powershell
(Invoke-WebRequest -Uri "https://loca.lt/mytunnelpassword").Content
```

Или через curl (если установлен):

```powershell
curl https://loca.lt/mytunnelpassword
```

## Что делать дальше

1. **Получите пароль** одним из способов выше
2. **Отправьте друзьям:**
   - URL туннеля: `https://tangy-ducks-leave.loca.lt` (ваш URL)
   - Пароль: `94.19.223.132` (ваш публичный IP)

3. **Друзья должны:**
   - Открыть URL в браузере
   - Ввести пароль (ваш публичный IP)
   - Нажать "Click to Submit"

## Важно

- Пароль = ваш публичный IP адрес (тот же, что показывает сервер при запуске)
- Пароль не меняется, пока не меняется ваш IP
- Если IP изменился, получите новый пароль тем же способом
