# Быстрый старт для работы с AI агентом

## 🚀 НАЧАЛО РАБОТЫ

Скопируй и вставь в новый чат с агентом:

```
Прочитай всю базу знаний в папке docs/. 

Есть задача: [ОПИШИ СВОЮ ЗАДАЧУ]

Используй workflow:
1. Читай docs/agent-onboarding.md
2. Создай спек в docs/specs/ используя spec-template.md
3. Задай мне вопросы
4. Проведи самооценку решения (риски, альтернативы)
5. Создай чеклист для самопроверки
6. Реализуй с тестами
```

---

## ✅ КОНЕЦ РАБОТЫ

Скопируй и вставь когда задача выполнена:

```
Завершай задачу. Проведи финальную проверку:

1. Чеклист из спека выполнен?
2. Все тесты написаны? (предложи запустить test_manual.py)
3. Соответствует patterns.md?
4. Логирование добавлено?
5. Нет hardcoded секретов?
6. Документация обновлена?
7. Линтер не находит ошибок?
8. Временные файлы удалены?

Подготовь для Git:
- git add [какие файлы]
- git commit -m "[что сделано]"
- git push
```
