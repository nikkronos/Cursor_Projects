# 🔧 Команды для проверки бота на сервере

**Вы уже подключены к серверу!** ✅

---

## 📊 Шаг 1: Проверьте статус бота

Выполните эту команду:

```bash
systemctl status tradetherapybot.service
```

**Нажмите `q` чтобы выйти из просмотра статуса.**

---

## 📋 Шаг 2: Посмотрите логи (если бот не работает)

Если бот не работает, посмотрите последние логи:

```bash
journalctl -u tradetherapybot.service -n 30 --no-pager
```

Это покажет последние 30 строк логов. Ищите ошибки (ERROR, CRITICAL).

---

## 🔄 Шаг 3: Попробуйте перезапустить

Если бот не работает, попробуйте перезапустить:

```bash
sudo systemctl restart tradetherapybot.service
```

Подождите 3 секунды и снова проверьте статус:

```bash
systemctl status tradetherapybot.service
```

---

## 🔍 Шаг 4: Проверьте файлы бота

```bash
cd /opt/tradetherapybot
ls -la
```

Проверьте:
- ✅ Есть ли файл `env_vars.txt`?
- ✅ Есть ли файл `bot.db`?
- ✅ Есть ли файл `main.py`?

---

## ⚡ Быстрая команда "всё сразу"

Если хотите увидеть всё сразу, выполните:

```bash
echo "=== СТАТУС БОТА ===" && systemctl status tradetherapybot.service --no-pager -l && echo -e "\n=== ПОСЛЕДНИЕ 20 СТРОК ЛОГОВ ===" && journalctl -u tradetherapybot.service -n 20 --no-pager
```
