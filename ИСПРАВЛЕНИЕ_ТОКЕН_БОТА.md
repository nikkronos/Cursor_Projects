# 🔧 Исправление: Бот не находит токен

**Проблема найдена!** Бот запущен, но не может найти токен.

---

## 🔍 Проблема

В логах видно:
```
WARNING - Bot token not found. Using mock bot for testing.
```

Это значит, что файл `env_vars.txt` либо отсутствует, либо в нём нет токена.

---

## ✅ Решение

### Шаг 1: Перейдите в папку бота

```bash
cd /opt/tradetherapybot
```

### Шаг 2: Проверьте файл env_vars.txt

```bash
cat env_vars.txt
```

**Что должно быть в файле:**
```
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_telegram_id
GROUP_CHAT_ID=id_группы
```

### Шаг 3: Если файл отсутствует или пустой - создайте/отредактируйте

```bash
nano env_vars.txt
```

**Добавьте (замените на ваши значения):**
```
BOT_TOKEN=ваш_токен_бота
ADMIN_ID=ваш_telegram_id
GROUP_CHAT_ID=id_группы
```

**Как сохранить в nano:**
1. Нажмите `Ctrl+O` (сохранить)
2. Нажмите `Enter` (подтвердить имя файла)
3. Нажмите `Ctrl+X` (выйти)

### Шаг 4: Проверьте права доступа

```bash
chmod 600 env_vars.txt
```

Это защитит файл от чтения другими пользователями.

### Шаг 5: Перезапустите бота

```bash
sudo systemctl restart tradetherapybot.service
```

### Шаг 6: Проверьте статус

```bash
systemctl status tradetherapybot.service
```

Теперь в логах НЕ должно быть предупреждения про токен!

---

## 📝 Где взять токен бота?

Если вы не помните токен:

1. Откройте Telegram
2. Найдите бота **@BotFather**
3. Отправьте `/mybots`
4. Выберите вашего бота
5. Нажмите **API Token**
6. Скопируйте токен
