# Проверка ошибок бота

## Команды для диагностики на сервере

```bash
ssh root@81.200.146.32

# 1. Проверить статус бота
systemctl status tradetherapybot

# 2. Проверить логи (последние 50 строк)
journalctl -u tradetherapybot -n 50 --no-pager

# 3. Проверить логи с ошибками
journalctl -u tradetherapybot -n 100 --no-pager | grep -i error

# 4. Попробовать запустить бота вручную для просмотра ошибок
cd /opt/tradetherapybot
python3 main.py
```

## Возможные проблемы

1. Синтаксическая ошибка в коде
2. Проблема с импортами
3. Ошибка в обработчике чеков (последние изменения)

## Если есть ошибка в коде

Нужно будет откатить изменения или исправить ошибку.
