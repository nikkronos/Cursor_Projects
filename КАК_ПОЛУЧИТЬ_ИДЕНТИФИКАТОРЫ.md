# 🔍 Как получить ADMIN_ID и GROUP_CHAT_ID

**Дата:** 02.12.2025

---

## 👤 Как получить ADMIN_ID (ваш Telegram ID)

### Способ 1: Через бота @userinfobot (самый простой)

1. Откройте Telegram
2. Найдите бота: **@userinfobot**
3. Отправьте ему любое сообщение (например, `/start`)
4. Он пришлёт ваш ID - это и есть **ADMIN_ID**

**Пример ответа:**
```
Your user ID: 123456789
```

Этот номер (123456789) и есть ваш ADMIN_ID.

---

### Способ 2: Через бота @RawDataBot

1. Откройте Telegram
2. Найдите бота: **@RawDataBot**
3. Отправьте `/start`
4. Он пришлёт JSON с вашими данными
5. Найдите там `"id"` - это ваш ID

---

### Способ 3: Если бот уже работал раньше

Если бот уже работал, ваш ID мог быть залогирован:
- В логах бота на сервере
- В базе данных бота

---

## 👥 Как получить GROUP_CHAT_ID (ID группы)

**Важно:** ID группы - это **отрицательное число**!

### Способ 1: Через бота @userinfobot (самый простой)

1. Добавьте бота **@userinfobot** в вашу группу
2. Сделайте его администратором (чтобы он мог видеть ID)
3. Отправьте ему сообщение в группе
4. Он пришлёт ID группы - это и есть **GROUP_CHAT_ID**

**Пример ответа:**
```
This chat ID: -1001234567890
```

Это отрицательное число (-1001234567890) и есть ваш GROUP_CHAT_ID.

---

### Способ 2: Временно через вашего бота

1. Добавьте вашего бота в группу
2. Сделайте его администратором
3. Временно добавьте в код логирование (или проверьте логи)
4. Когда кто-то напишет в группу, в логах появится ID группы

---

### Способ 3: Если группа уже работала с ботом

Если группа уже была настроена:
- Проверьте логи бота на сервере
- Проверьте файл конфигурации (если сохраняли)

---

## 📝 Пример заполнения env_vars.txt

После получения всех данных, файл должен выглядеть так:

```
BOT_TOKEN=8253836050:AAHXl5bPETY2AU6--lXVNgwbbSdA0fpBeuk
ADMIN_ID=123456789
GROUP_CHAT_ID=-1001234567890
```

**Замените:**
- `123456789` на ваш реальный ADMIN_ID
- `-1001234567890` на ваш реальный GROUP_CHAT_ID

---

## ⚡ Быстрый способ (если группа уже есть)

Если группа уже работает и вы не помните ID:

1. **ADMIN_ID**: Используйте @userinfobot - быстро и просто
2. **GROUP_CHAT_ID**: Добавьте @userinfobot в группу - он покажет ID

---

## ✅ После получения данных

1. Заполните файл `env_vars.txt` на сервере
2. Перезапустите бота
3. Готово! 🎉
