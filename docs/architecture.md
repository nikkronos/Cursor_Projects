# Архитектура проекта

## Стек технологий
- **Язык:** Python 3.x
- **Библиотека бота:** `pyTelegramBotAPI` (telebot)
- **База данных:** PostgreSQL (`psycopg2`)
- **Планировщик задач:** `APScheduler` (для проверки подписок в фоне)
- **Конфигурация:** `python-dotenv` (загрузка переменных окружения)

## Структура папок
```
/
├── bot.py                # Основной файл запуска и логики бота
├── config.py             # Загрузка конфигурации
├── env_vars.txt          # Файл с переменными окружения (Токены, БД) - НЕ В GIT!
├── .gitignore            # Исключения для Git
├── docs/                 # Документация проекта
│   ├── project.md
│   ├── git-workflow.md
│   └── ...
└── logs/                 # Логи работы бота
    └── bot.log
```

## Внешние интеграции
- **Telegram Bot API:** Взаимодействие с пользователями и управление группой.
- **PostgreSQL Server (Timeweb):** Хранение данных пользователей и чеков.
