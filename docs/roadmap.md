# Roadmap (План развития)

## Ближайшие задачи
- [ ] Развертывание бота на постоянном сервере (чтобы работал 24/7 без включенного ПК).
- [ ] Настройка автоматического CI/CD (автоматическое обновление на сервере при пуше в GitHub).

## Идеи на будущее
- [ ] Интеграция с платежной системой (ЮКасса/Robokassa) для авто-оплаты без ручной проверки чеков.
- [ ] Личный кабинет пользователя с историей оплат.
- [ ] Реферальная система (пригласи друга).
- [ ] Рассылка по всем пользователям через админку.
