# Функционал (Features)

## Статусы:
- [x] Готово

## Список функций

### Пользовательский интерфейс
- [x] Команда `/start` и регистрация в БД.
- [x] Главное меню (Тарифы, Правила, О Нас, Обратная связь, Статус, Отзывы, Публичная оферта).
- [x] Просмотр тарифов и реквизитов для оплаты (выбор тарифа).
- [x] Отправка чека (фото/документ) для подтверждения оплаты.
- [x] Ветка "Не буду платить" с опросом причин (Нет денег / Не торгую / Другое).
- [x] Получение уведомлений об истечении подписки.
- [x] Раздел "Отзывы" со ссылкой на канал и скриншотом.
- [x] Раздел "О Нас" с описанием и скриншотами.

### Административная панель
- [x] Меню `/admin`.
- [x] Просмотр всех участников (активные/бывшие).
- [x] Ручное добавление/вычитание дней подписки.
- [x] Полное удаление пользователя (кик + сброс подписки).
- [x] Подтверждение/Отклонение присланных чеков.
- [x] Просмотр истории чеков (последние 10).
- [x] Очистка истории чеков.
- [x] Команда `/backup` для получения файла базы данных.

### Автоматизация и Логика
- [x] Фоновая задача (раз в 3 мин) для проверки сроков подписки.
- [x] Автоматический кик пользователей с истекшей подпиской.
- [x] Защита от входа по ссылке без активной подписки (Failsafe kick).
- [x] Логирование ошибок и действий в файл.
