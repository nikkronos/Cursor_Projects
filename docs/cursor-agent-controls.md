# Управление изменениями агента в Cursor

## Undo All / Keep All / Review

Когда агент вносит изменения в файлы, Cursor показывает панель с тремя кнопками:

### 1. **Undo All** (Отменить всё)
- Отменяет **все** изменения, которые агент предложил
- Возвращает файлы в исходное состояние
- Используйте, если изменения не подходят

### 2. **Keep All** (Сохранить всё)
- Применяет **все** изменения, которые агент предложил
- Сохраняет все файлы с изменениями
- Используйте, если все изменения подходят

### 3. **Review** (Просмотр)
- Позволяет просмотреть изменения перед применением
- Можно выбрать, какие изменения применить, а какие отклонить
- Используйте для выборочного применения изменений

## Автоматическое применение

**В режиме Agent (текущий режим):**
- ✅ Агент **может применять изменения автоматически** через инструменты редактирования
- ✅ Изменения применяются напрямую в файлы
- ✅ Можно откатить через Git: `git checkout -- <файл>` или `git reset HEAD~1`
- ✅ Можно использовать Undo в редакторе (Ctrl+Z)

**В режиме Ask:**
- ❌ Агент только **предлагает** изменения
- ❌ Вы должны **вручную** выбрать: Undo All, Keep All или Review

## Рекомендация

**Для текущей работы (Agent режим):**
- Агент применяет изменения автоматически
- Всегда можно откатить через Git
- Проверяйте изменения перед коммитом: `git diff`

**Для безопасности:**
- Регулярно делайте коммиты: `git commit`
- Используйте `git status` для проверки изменений
- При необходимости откатывайте: `git reset HEAD~1` (откат последнего коммита)
