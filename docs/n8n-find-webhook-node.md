# Как найти ноду Webhook в N8N

## Способ 1: Поиск по названию

1. В редакторе workflow нажмите на **"Add first step..."** (большой "+" в центре)
2. В открывшейся панели поиска введите: **"webhook"**
3. Выберите ноду **"Webhook"** из результатов поиска

## Способ 2: Через категории

1. Нажмите на **"Add first step..."**
2. В левой панели найдите категорию **"Core Nodes"** или **"Trigger Nodes"**
3. Прокрутите список и найдите **"Webhook"**

## Способ 3: Использовать AI для создания

1. Нажмите на **"Build with AI"** (иконка с волшебной палочкой)
2. Введите запрос: **"Create a webhook that receives POST requests"**
3. AI создаст workflow с webhook нодой автоматически

## Способ 4: Через шаблон

1. Вернитесь на главную страницу workflows (Overview)
2. Найдите шаблон **"Lead Gen Agent (Telegram)"** или любой другой с webhook
3. Используйте шаблон как основу

## Визуальное описание

После нажатия "Add first step..." вы увидите:
- **Панель поиска** вверху - введите "webhook"
- **Категории нод** слева - ищите в "Core Nodes" или "Trigger Nodes"
- **Список нод** в центре - прокрутите до "Webhook"

## Если не можете найти

Попробуйте:
1. Обновить страницу (F5)
2. Проверить, что вы в редакторе workflow (не на главной странице)
3. Использовать AI: "Build with AI" → "webhook workflow"
