## Важная информация о деплое

### Текущее состояние:
- ✅ Бот развернут на сервере: `81.200.146.32`
- ✅ Путь на сервере: `/opt/tradetherapybot`
- ✅ Systemd service: `tradetherapybot.service`
- ✅ GitHub Actions workflows настроены (тесты + деплой)

### Автоматический деплой:
- **Статус:** GitHub Actions workflows созданы, но требуется настройка SSH ключей и GitHub Secrets
- **Инструкция:** См. `docs/CICD_SETUP.md` или `QUICK_START_CICD.md`
- **После настройки:** Каждый `git push origin main` автоматически обновит бота на сервере

### Ручной деплой (если автоматический не настроен):
```bash
ssh root@81.200.146.32
cd /opt/tradetherapybot
git pull origin main
pip3 install --break-system-packages -r requirements.txt
sudo systemctl restart tradetherapybot.service
```

**Важно:** Всегда проверяй, что бот работает после обновления!

### Предотвращение потери файлов:
- **Главное правило:** Коммитьте часто! После каждого значимого изменения
- Регулярно делайте `git push origin main` - GitHub служит резервной копией
- Проверяй файлы перед коммитом - нет ли пустых файлов
- Подробнее: `docs/preventing-file-loss.md` и `docs/git-workflow.md`
