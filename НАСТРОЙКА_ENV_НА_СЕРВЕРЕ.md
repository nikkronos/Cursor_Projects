# 🔧 Настройка env_vars.txt на сервере

**Важно:** Этот файл содержит секретные данные! Не коммитьте его в Git!

---

## 📝 Что нужно сделать

### Шаг 1: Подключитесь к серверу

```bash
ssh root@81.200.146.32
```

### Шаг 2: Перейдите в папку бота

```bash
cd /opt/tradetherapybot
```

### Шаг 3: Создайте файл env_vars.txt

```bash
nano env_vars.txt
```

### Шаг 4: Добавьте эти строки (замените значения!)

```
BOT_TOKEN=8253836050:AAHXl5bPETY2AU6--lXVNgwbbSdA0fpBeuk
ADMIN_ID=ваш_telegram_id
GROUP_CHAT_ID=id_вашей_группы
```

### Шаг 5: Сохраните файл

В nano:
1. Нажмите `Ctrl+O` (сохранить)
2. Нажмите `Enter` (подтвердить)
3. Нажмите `Ctrl+X` (выйти)

### Шаг 6: Защитите файл

```bash
chmod 600 env_vars.txt
```

### Шаг 7: Перезапустите бота

```bash
sudo systemctl restart tradetherapybot.service
```

## 🔍 Как получить ADMIN_ID

### Способ 1: Через бота (если бот уже работал раньше)

1. Откройте Telegram
2. Найдите бота **@userinfobot**
3. Отправьте ему любое сообщение
4. Он пришлёт ваш ID

Или:

1. Найдите бота **@RawDataBot**
2. Отправьте `/start`
3. Он пришлёт ваш ID

### Способ 2: Через вашего бота (если он работал)

Когда бот работал, он должен был логировать ваш ID при команде `/start`.

---

## 🔍 Как получить GROUP_CHAT_ID

### Способ 1: Если группа уже создана

1. Добавьте бота в группу
2. Добавьте бота **@userinfobot** в группу
3. Отправьте ему сообщение в группе
4. Он пришлёт ID группы (будет отрицательное число, например `-1001234567890`)

### Способ 2: Через бота

1. Добавьте бота в группу
2. Сделайте его администратором
3. В логах бота будет ID группы (когда кто-то напишет)

### Способ 3: Если группа уже работала с ботом

Проверьте логи бота - там должен быть ID группы.

3. Попробуйте отправить `/start` боту

Бот должен заработать! 🎉
