## 📚 Важная информация

### Как работает автоматический деплой:
1. При `git push origin main` автоматически запускаются тесты
2. Если тесты прошли → автоматически запускается деплой
3. Создается бэкап БД
4. Обновляется код через `git pull`
5. Обновляются зависимости
6. Перезапускается systemd service
7. При ошибках происходит автоматический откат

### GitHub Secrets (настроены):
- `SSH_PRIVATE_KEY` - приватный SSH ключ
- `SSH_HOST` - IP адрес сервера
- `SSH_PORT` - SSH порт
- `SSH_USER` - пользователь SSH
- `DEPLOY_PATH` - путь к проекту на сервере
- `SERVICE_NAME` - имя systemd service
