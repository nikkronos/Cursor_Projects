# ✅ Добавить ADMIN_ID и GROUP_CHAT_ID

**Токен уже настроен! Теперь нужно добавить ID.**

---

## 👤 Шаг 1: Получить ADMIN_ID (ваш Telegram ID)

### Способ 1: Через @userinfobot (самый простой)

1. Откройте Telegram на телефоне/компьютере
2. Найдите бота: **@userinfobot**
3. Отправьте ему любое сообщение (например, `/start`)
4. Он пришлёт ваш ID

**Пример ответа:**
```
Your user ID: 123456789
```

Этот номер и есть ваш **ADMIN_ID**.

---

### Способ 2: Если бот уже работал

В логах бота уже виден ID пользователя: `135366416`

Если это ваш ID - используйте его!

---

## 👥 Шаг 2: Получить GROUP_CHAT_ID (ID группы)

### Способ 1: Через @userinfobot

1. Добавьте бота **@userinfobot** в вашу группу
2. Сделайте его администратором (чтобы он мог видеть ID)
3. Отправьте ему сообщение в группе
4. Он пришлёт ID группы

**Пример ответа:**
```
This chat ID: -1001234567890
```

Это отрицательное число и есть ваш **GROUP_CHAT_ID**.

---

### Способ 2: Если группа уже работает с ботом

Если группа уже была настроена, ID может быть в логах или базе данных.

---

## 📝 Шаг 3: Добавить в файл на сервере

После получения ID, отредактируйте файл:

```bash
cd /opt/tradetherapybot
nano env_vars.txt
```

Добавьте полученные значения:

```
BOT_TOKEN=8253836050:AAHXl5bPETY2AU6--lXVNgwbbSdA0fpBeuk
ADMIN_ID=ваш_id_от_userinfobot
GROUP_CHAT_ID=-id_группы_от_userinfobot
```

**Сохраните:**
- `Ctrl+O` (сохранить)
- `Enter` (подтвердить)
- `Ctrl+X` (выйти)

---

## 🔄 Шаг 4: Перезапустить бота

```bash
sudo systemctl restart tradetherapybot.service
```

---

## ✅ Проверка

```bash
systemctl status tradetherapybot.service
```

Теперь бот должен работать полностью!
