## Команды для Git:

### 1. Инициализация репозитория (если еще не инициализирован):

```bash
git init
```

### 2. Добавить все файлы:

```bash
git add .
```

### 3. Проверить, что будет добавлено:

```bash
git status
```

Убедитесь, что НЕ добавляются:
- `bot.db` (база данных)
- `logs/` (логи)
- `__pycache__/` (кэш)
- `env_vars.txt` (секретные данные)

### 4. Создать первый коммит:

```bash
git commit -m "Initial commit: Trade Therapy Bot v2.1.0"
```

### 5. Добавить удаленный репозиторий (если есть):

```bash
git remote add origin <url_вашего_репозитория>
```

### 6. Отправить в репозиторий:

```bash
git push -u origin main
```

или

```bash
git push -u origin master
```
