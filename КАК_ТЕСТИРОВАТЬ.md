# Как запустить тесты вручную

## Способ 1: Через терминал в Cursor/VS Code

1. **Откройте терминал:**
   - В Cursor/VS Code нажмите `Ctrl + ~` (или `Ctrl + Ё` на русской раскладке)
   - Или меню: `Terminal` → `New Terminal`

2. **Перейдите в папку проекта:**
   ```bash
   cd TradeTherapyBot
   ```

3. **Запустите тест:**
   ```bash
   python test_manual.py
   ```

## Способ 2: Через командную строку Windows

1. **Откройте командную строку (cmd) или PowerShell:**
   - Нажмите `Win + R`
   - Введите `cmd` или `powershell`
   - Нажмите Enter

2. **Перейдите в папку проекта:**
   ```bash
   cd "C:\Users\krono\OneDrive\Рабочий стол\Cursor_test\TradeTherapyBot"
   ```
   (Путь может быть другим, смотрите где у вас проект)

3. **Запустите тест:**
   ```bash
   python test_manual.py
   ```

## Способ 3: Прямо из Cursor (самый простой)

1. В Cursor откройте файл `test_manual.py`
2. Нажмите правой кнопкой мыши на файле
3. Выберите "Run Python File in Terminal"
4. Или используйте горячую клавишу (обычно `F5` или `Ctrl+F5`)

## Если не работает

1. **Ошибка "python не найден":**
   - Убедитесь, что Python установлен
   - Попробуйте `python3` вместо `python`
   - Или `py test_manual.py`

2. **Ошибка "модуль не найден":**
   - Убедитесь, что вы в папке `TradeTherapyBot`
   - Проверьте, что файл `test_manual.py` находится в этой папке
