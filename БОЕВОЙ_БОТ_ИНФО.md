## 🚀 Быстрая настройка

### 1. Получить токен
```
@BotFather → /mybots → @tradetherapybot → API Token
```

### 2. Настроить на сервере
```bash
ssh root@81.200.146.32
cd /opt/tradetherapybot
nano env_vars.txt
```

**Вставить:**
```
BOT_TOKEN=токен_от_BotFather
ADMIN_ID=ваш_id
GROUP_CHAT_ID=id_группы
```

**Перезапустить:**
```bash
systemctl restart tradetherapybot.service
```

### 3. Проверить
- Найти @tradetherapybot
- Отправить `/start`
- Проверить `/admin`
